Опции
-----

Имена опций:

* ``"-f"``, ``"--foo-bar"``, имя параметра функции будет ``"foo_bar"``
* ``"-x"``, имя ``"x"``
* ``"-f"``, ``"--filename"``, ``"dest"``, имя ``"dest"``
* ``"--CamelCase"``, имя ``"camelcase"``
* ``"-f"``, ``"-fb"``, имя ``"f"``
* ``"--f"``, ``"--foo-bar"``, имя ``"f"``
* ``"---f"``, имя ``"_f"``
